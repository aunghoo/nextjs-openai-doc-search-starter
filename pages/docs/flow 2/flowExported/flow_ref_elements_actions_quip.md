# Quip Flow Core Actions

Quip provides several core actions for organizing, creating, and copying your Quip content in flows. To add one of these actions to your flow, add an Action element. Then select the **Quip** category, and search for the appropriate action.

|Available in: **Lightning Experience**|

**Important:** Quip core actions don’t support system-run flows or automated process users. Quip core actions execute in the context of the user, who is also referred to as the context user. The flow has access to whatever the context user has access to.

-   **[Flow Core Actions for Quip: Create Quip Document](../flow/flow_ref_elements_actions_quip_create_doc.md)**  
Create a document, spreadsheet, or slide deck, and add content to it.
-   **[Flow Core Action for Quip: Create Quip Chat](../flow/flow_ref_elements_actions_quip_create_chat.md)**  
Create a chat room, and send a message to its members.
-   **[Flow Core Action for Quip: Create Quip Folder](../flow/flow_ref_elements_actions_quip_create_folder.md)**  
Create a private folder, or add it to existing folders.
-   **[Flow Core Action for Quip: Copy Quip Document](../flow/flow_ref_elements_actions_quip_copy_doc.md)**  
To use a document as a template, create a copy. By default, copied documents are added to the running user’s Private folder in Quip.
-   **[Flow Core Action for Quip: Copy Quip Content \(Retired\)](../flow/flow_ref_elements_actions_quip_copy_content.md)**  
Copy content from a source slide deck, and paste it in a target slide.
-   **[Flow Core Action for Quip: Copy with Live Paste](../flow/flow_ref_elements_actions_quip_live_paste.md)**  
Copy content from a source document, and paste it with Live Paste in a new document. When the source content is updated, documents with the live pasted content stay up to date.
-   **[Flow Core Action for Quip: Attach Quip Document to Record](../flow/flow_ref_elements_actions_quip_attach_doc_record.md)**  
Attach a document to a Salesforce record. Linked documents show up in the Quip Associated Documents component.
-   **[Flow Core Action for Quip: Edit Quip Document](../flow/flow_ref_elements_actions_quip_edit_doc.md)**  
Edit content in a document, spreadsheet, or slide. Add or replace content based on a document section.
-   **[Flow Core Action for Quip: Lock Quip Document](../flow/flow_ref_elements_actions_quip_lock_doc.md)**  
To mark a document as complete, lock document edits.
-   **[Flow Core Action for Quip: Lock Quip Section](../flow/flow_ref_elements_actions_quip_lock_section.md)**  
To mark sections of a document as complete or to keep them safe from accidental edits, lock them.
-   **[Flow Core Action for Quip: Export Quip Document to PDF](../flow/flow_ref_elements_actions_quip_export_to_pdf.md)**  
To mark a document as complete and to keep a copy for your records, export it as a PDF. You can attach the exported PDF to a document or to a Salesforce record.
-   **[Flow Core Action for Quip: Send Message in Quip Chat](../flow/flow_ref_elements_actions_quip_send_message_chat.md)**  
Send a message in a chat room.
-   **[Flow Core Action for Quip: Send a Message in a Document](../flow/flow_ref_elements_actions_quip_send_message_doc.md)**  
Add a message to the conversations pane of a document.
-   **[Flow Core Action for Quip: Add Quip Document to Folder](../flow/flow_ref_elements_actions_quip_add_doc_folder.md)**  
Add a document to a folder to organize and share your documents.
-   **[Flow Core Action for Quip: Add Members to Document](../flow/flow_ref_elements_actions_quip_add_members_doc.md)**  
Add members with different levels of access to a document.
-   **[Flow Core Action for Quip: Add Members to Quip Chat](../flow/flow_ref_elements_actions_quip_add_members_chat.md)**  
Add users to a chat room.
-   **[Flow Core Action for Quip: Remove Quip Document from Folder](../flow/flow_ref_elements_actions_quip_remove_from_folder.md)**  
Remove a document from a folder. Make a shared document private again.
-   **[Flow Core Action for Quip: Remove Members from Quip Document](../flow/flow_ref_elements_actions_quip_remove_from_doc.md)**  
To rescind access to a document for certain users, remove them from the document.
-   **[Flow Core Action for Quip: Remove Members from Quip Chat](../flow/flow_ref_elements_actions_quip_remove_from_chat.md)**  
Remove users from a chat room.
